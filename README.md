# 🎯 Vision

ResumeShastra aims to simplify resume creation for students and freshers.

Instead of spending hours adjusting templates, users simply fill guided forms and generate a professional ATS-friendly resume within minutes.

> Less Typing. More Tapping.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yaswanthkumar-07/resume-shastra.git
```

---

## Backend

```bash
cd backend
./mvnw spring-boot:run
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```
